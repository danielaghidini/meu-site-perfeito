# Meu Site Perfeito

## Descrição

Desenvolvimento Web com React e CMS Headless.

## Tecnologias

Este projeto foi construído com:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Como rodar localmente

Certifique-se de ter o Node.js instalado.

```sh
# Passo 1: Instalar dependências
npm install

# Passo 2: Iniciar o servidor de desenvolvimento
npm run dev
```

## Build para Produção

```sh
npm run build
```
